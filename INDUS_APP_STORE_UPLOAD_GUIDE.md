## 🐛 TROUBLESHOOTING

### Issue: APK Won't Upload
**Solution**: 
- Check file size < 500MB
- Ensure Android target SDK >= 31
- Verify APK is signed

### Issue: App Rejected for Permissions
**Solution**:
- Indus might flag excessive permissions
- Check what's needed vs requested
- Rebuild with minimal permissions

### Issue: App Gets Low Ratings
**Solution**:
- Fix bug immediately
- Release quick update (v1.0.1)
- Respond to reviews
- Ask users to update

### Issue: Need to Update App
**Solution**:
1. Build new APK (v1.0.1)
2. Go to Indus Dashboard
3. Click "Upload New Version"
4. Upload new APK
5. Approval again (usually faster)
